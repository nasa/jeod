98 /**
99  * Class DynBody is the base class for all dynamic bodies.
100  * A DynBody is a MassBody that is connected to the outside world.
101  * These connections are in the form of three reference frames tied to the
102  * body -- the structural, core body, and composite body frames.
103  *
104  * For a non-root body, the states for each of these frames is calculated
105  * based on the parent body's state and on the body attachment.
106  *
107  * For a root body, one of these three frames must be integrated.
108  * The details of how that integration is performed is the subject of classes
109  * that derive from DynBody.
110  */
