100 /**
101  * Class DynBody is the base class for all dynamic bodies.
102  * A DynBody is a MassBody that is connected to the outside world.
103  * These connections are in the form of three reference frames tied to the
104  * body -- the structural, core body, and composite body frames.
105  *
106  * For a non-root body, the states for each of these frames is calculated
107  * based on the parent body's state and on the body attachment.
108  *
109  * For a root body, one of these three frames must be integrated.
110  * The details of how that integration is performed is the subject of classes
111  * that derive from DynBody.
112  */
