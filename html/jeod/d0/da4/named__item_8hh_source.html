79 /**
80  * Provides a set of static methods for constructing dot-conjoined names.
81  * The methods defined in this class allocate memory and do not release it.
82  * Releasing that memory is the responsibility of the calling function.
83  * Use the macro JEOD_DELETE_ARRAY to release this memory.
84  *
85  * Prior to JEOD 4.0, the NamedItem class was not instantiable. It is in
86  * JEOD 4.0. The NamedItem class forms the basis of a thing with a name,
87  * with the name being a std::string. The construct_name functions and
88  & related functions that allocate a C-style string are deprecated.
