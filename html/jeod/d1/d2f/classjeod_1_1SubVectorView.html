Detailed Description

template<class UnderlyingType, typename ElemType, typename RefType, typename BaseRefType>
class jeod::SubVectorView< UnderlyingType, ElemType, RefType, BaseRefType >

Provides a view into a part of a vector.

Template Parameters
UnderlyingTypeThe type of the underlying object. This must, at a minimum, provide an index operator and a size method.
ElemTypeThe type of the elements held by the underlying object. This is assumed to be a primitive type.
RefTypeNominally, a reference to ElemType. This should be ElemType in the case of a const SubVectorView so as to preclude alterations of the underlying object.
BaseRefTypeAlso nominally a reference to ElemType. This should also be ElemType in the case of a const SubVectorView so as to preclude alterations of the underlying object.

Parameters
obj_inThe object that underlies this view.
rangeA begin, end pair that denote the extent of the view. The begin value corresponds to this view's zeroth element. The end value is one past the last element indexed by this view. To be valid, the end value must not be less than the begin value and must not be greater than the underlying vector's size.
Exceptions
