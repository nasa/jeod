Detailed Description

The BaseOptionValue is the base class for all option descriptor classes.

All instantiable classes that derive from BaseOptionValue will have

  • Definitions of the three pure virtual methods declared below. – takes_argument() indicates whether the option takes an argument, – set_default() sets the default value for the option, and – parse_value() sets the option value based on the argument.
  • A reference to a target variable. CmdlineParser::parse() calls the set_default() method for each registered option to set the option's target to the default value. When CmdlineParser::parse() recognizes an option on the command line, it calls the option's parse_value() method to set the option's target.
