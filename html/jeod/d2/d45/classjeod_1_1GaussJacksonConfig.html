unsigned int jeod::GaussJacksonConfig::max_correction_iterations {10}

Maximum number of correction steps allowed before the integrator is deemed to be not converging.

The algorithm is run in predict-only mode if this limit is zero. The corrector is applied but once with the limit is one. A one-time warning is issued if the limit is 2 or more and if the the algorithm would make more corrections were it not for this limit. Defaults to 10.trick_units(–)
