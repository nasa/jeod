98 /**
99  * Represent both an atomic chunk of mass and an interconnected set of masses.
100  * Multiple MassBody objects can be attached to one another in a tree structure.
101  * This connected sets of bodies has a composite set of mass properties.
102  * Sans these connections, a body will have some core mass proprties.
103  *
104  * The MassBody class is one of the key classes in JEOD 2.0. In addition to
105  * representing masses, it is also the parent of the DynBody class, which is
106  * used in JEOD 2.0 to represent vehicles.
107  *
108  * \par Assumptions and Limitations
109  * - Rigid Bodies
110  * - Tree attachment Structure
111  */
