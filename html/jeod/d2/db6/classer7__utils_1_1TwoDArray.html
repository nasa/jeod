Detailed Description

template<typename T>
class er7_utils::TwoDArray< T >

RAII template class that implements a rectangular two dimensional array.

The implementation provides two special-purpose features that are needed by some of the ER7 utilities and JEOD integrators. The rows of the array can be rotated and downsampled.

TType of each element of the array.
