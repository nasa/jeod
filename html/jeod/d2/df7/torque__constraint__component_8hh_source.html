67 /**

69  * the application of a torque.

111  /**
112  * Prepare this object for solving a constraint problem.
113  *
114  * The direction in which the constraint torque acts is known at this point,
115  * but the magnitude is not. This function sets the torque to a unit torque
116  * in the known constraint direction. This enables the computation of
117  * interactions with other constraints (the A matrix in A*x=b) and the
118  * computation of the constraint response (the b vector in A*x=b).
119  * The solver subsequently solves A*x=b, which yields the torque magnitude.
120  *
121  * @param vehicle_properties Various vehicle properties.
122  * @param non_grav_state The non-gravitational response of the
123  * root body to non-constraint forces and torques, including
124  * the pre-constraint wrenches.
125  */

127  const VehicleNonGravState & non_grav_state) override;
