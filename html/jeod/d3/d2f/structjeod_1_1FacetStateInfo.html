Detailed Description

This is a structure used only in the surface model to aid in relative state calculations for articulation.

For each mass body that needs a relative state calculated w.r.t. the mass body named in struct_body_name, one of these objects will be instantiated. That way, the relative state information must only be calculated once per mass body.
