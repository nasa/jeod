70 /**
71  * Represents a linear system of equations, A*x = b, where A is conceptually
72  * an NxN matrix, and x and b are conceptually N-vectors. This class
73  * solves for x in A*x = b. How it does that is up to a derived class.
74  * `solve` is pure virtual in this base class template.
75  *
76  * An implementor of an instantiable derived class of this class template
77  * must implement the `solve` method.
78  *
79  * A user of this class template
80  * - Sets the number of degrees of freedom N,
81  * - Populates the A matrix and b vector, and
82  * - Requests the object to solve for x.
83  *
84  * Users must not assume that `solve` leaves the populated A matrix and b vector
85  * untouched. To the contrary, an instantiable solver might well modify these
86  * elements in the process of solving the matrix equation. It is up to the user
87  * to fully populate the A matrix and b vector prior to calling `solve`.
88  */
