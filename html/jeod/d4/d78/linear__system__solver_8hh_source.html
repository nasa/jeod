68 /**
69  * Represents a linear system of equations, A*x = b, where A is conceptually
70  * an NxN matrix, and x and b are conceptually N-vectors. This class
71  * solves for x in A*x = b. How it does that is up to a derived class.
72  * `solve` is pure virtual in this base class template.
73  *
74  * An implementor of an instantiable derived class of this class template
75  * must implement the `solve` method.
76  *
77  * A user of this class template
78  * - Sets the number of degrees of freedom N,
79  * - Populates the A matrix and b vector, and
80  * - Requests the object to solve for x.
81  *
82  * Users must not assume that `solve` leaves the populated A matrix and b vector
83  * untouched. To the contrary, an instantiable solver might well modify these
84  * elements in the process of solving the matrix equation. It is up to the user
85  * to fully populate the A matrix and b vector prior to calling `solve`.
86  */
