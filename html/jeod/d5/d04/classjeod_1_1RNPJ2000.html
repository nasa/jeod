◆ initialize()

void jeod::RNPJ2000::initialize ( DynManager &  dyn_manager)
overridevirtual

Apply the various options, including fidelity of RNP and if polar motion is enabled or not, then call the base class initializer.

Note that this function will still run even if the active flag is set to false. Additionally, note that this function does not update the rotation matrix between inertial and planet fixed. It only sets up the object to work with the DynManager. update_rnp must be called to update the RNP matrix. This function will also cache of TimeGMST from the given TimeManager

Parameters
[in,out]dyn_managerDynManager where the planet attitude to be updated is contained
