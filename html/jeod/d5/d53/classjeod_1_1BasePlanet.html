Detailed Description

A BasePlanet contains the base data needed to model a planet in JEOD.

A BasePlanet has a name, a planet-centered inertial reference frame, and a planet-centered planet-fixed reference frame. Details of the planet's shape and mass are in the Planet class, which derives from BasePlanet.
