Detailed Description

template<class ParentType, typename ElemType, typename RefType>
class jeod::SubArrayView< ParentType, ElemType, RefType >

Represents a subarray view into an array.

The function call operator provides unprotected access to an element of the subarray. The member function at() ensures that the indexed element is valid.
