◆ initialize()

void jeod::PlanetOrientation::initialize ( DynManager &  dyn_manager)
virtual

Goes to the dyn manager given and searches for the planet indicated by the user inputted name, in the given dyn manager.

Will send a fail message if the named planet is not found. Additionally, this will register this ephemeris interface with the given DynManager, and will set up the contained EphemerisOrientation to control the named planet.

Parameters
[in,out]dyn_managerDynManager where the attitude will be applied
