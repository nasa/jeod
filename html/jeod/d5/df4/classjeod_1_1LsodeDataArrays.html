◆ lin_alg

double** jeod::LsodeDataArrays::lin_alg {}

was RWORK[LWM:LWM+LENWM-1].

LWM = LYH + (NYH*(MAXORD+1)) lin_alg_1 = rwork[lwm] lin_alg_2 = rwork[lwm + 1] lin_alg[i,j] = rwork[lwm+ j*n + i + 2]. The first two elements are treated differently, then it goes to an array that is sized based on the correction_method. The array sizes are as follows, ordered by value of correction_method: 0: 0 1,2: n x n 3: 1 x n 4,5: (2*ml+mu+1) x n.trick_units(–)
