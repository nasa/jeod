Detailed Description

The S_define-level class that provides planetary ephemerides.

The SpiceEphemeris class constructs the ephemeris reference frame tree and updates the states of the planets based on data from a SPICE file or files.

◆ inactive

bool jeod::SpiceEphemeris::inactive {}
protected

If set to true, makes the model inactive.

This is set to false (i.e., model is active) by the constructor. Setting this flag to true prior to initialization time will result in the model never doing anything. Setting this flag to true after the model has been active for some time turns it off from that point forward.
