80 /**
81  * This is the base class for the JEOD replacements of the STL sequence
82  * containers. The class derives from JeodSTLContainer, the base class for
83  * the JEOD replacements of the STL containers.
84  *
85  * A key goal of the JEOD STL sequence container replacement effort is to
86  * provide checkpointable replacements that transparently provide the full
87  * functionality of the ISO/IEC 14882:2003 STL sequence containers.
88  * This class begins that effort by defining types and member functions
89  * common to the STL deque, list, and vector class templates.
90  * Non-common methods are the responsibility of derived class templates
91  * specialized to a specific container types.
92  *
93  * @note Exceptions to full functionality goal:
94  * The above goal is not and never will be fully achieved. Exceptions are:
95  * - JEOD doesn't supply a replacement for std::deque.
96  * JEOD doesn't use deques.
97  * - The full panoply of STL sequence container constructors is not supplied.
98  */
