81 /**
82  * Describe a frame of reference and define operations on reference frames.
83  * A JEOD reference frame
84  * - Is characterized by an origin and and a set of three orthogonal axes.
85  * - Provides a mechanism for specifying the translational and rotational
86  * states of an object in space (particularly, Cartesian three space).
87  * - Is itself an object whose translational and rotational states can be
88  * specified/determined in terms of some other reference frame.
89  * - Is a node in a rooted tree of reference frames, each of which has some
90  * specific state with respect to another node in the tree.
91  * - Can be active (or inactive). An active frame supposedly will have a
92  * (fairly) current state. All bets are off if the frame is inactive.
93  * - Can have subscribers, which are external entities that for some reason
94  * need the frame to be active.
95  *
96  * Reference frames are one of the key concepts that define JEOD 2.0.
97  */
