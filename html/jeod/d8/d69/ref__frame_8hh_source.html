82 /**
83  * Describe a frame of reference and define operations on reference frames.
84  * A JEOD reference frame
85  * - Is characterized by an origin and and a set of three orthogonal axes.
86  * - Provides a mechanism for specifying the translational and rotational
87  * states of an object in space (particularly, Cartesian three space).
88  * - Is itself an object whose translational and rotational states can be
89  * specified/determined in terms of some other reference frame.
90  * - Is a node in a rooted tree of reference frames, each of which has some
91  * specific state with respect to another node in the tree.
92  * - Can be active (or inactive). An active frame supposedly will have a
93  * (fairly) current state. All bets are off if the frame is inactive.
94  * - Can have subscribers, which are external entities that for some reason
95  * need the frame to be active.
96  *
97  * Reference frames are one of the key concepts that define JEOD 2.0.
98  */
