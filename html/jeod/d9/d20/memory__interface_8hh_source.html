201  /**
202  * Get the simulation engine's name (if any) of the address.
203  * A derived class associated with a simulation engine that does not support
204  * this translation should return an empty string for all calls.
205  * When the underlying simulation engine does support this translation,
206  * the implementation should return values as follows:
207  * - The string "NULL" if the input address is the null pointer.
208  * - The empty string to indicate an invalid input address or an input
209  * address that is unknown to the simulation engine.
210  * - A non-empty, non-"NULL" string to indicate a valid address.
211  * Applying the get_address_at_name method to this result must yield
212  * the input address.
213  * \note
214  * A stubbed implementation should have its is_checkpoint_restart_supported
215  * method return false.
216  * \return Name of the address, if any
217  * \param[in] addr Address of memory to identified by name
