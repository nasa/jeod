Detailed Description

template<class ArrayType, typename ElemType, typename RefType>
class jeod::ColumnView< ArrayType, ElemType, RefType >

Represents a view into a column of an array.

The index operator provides unprotected access to an element of the view. The member function at() ensures that the indexed element is valid.

Template Parameters
ArrayTypeThe type of the underlying object. This must, at a minimum, provide an function call operator that accesses array elements and a size method that returns a pair.
ElemTypeThe type of the elements held by the underlying object. This is assumed to be a primitive type.

Note
This can create a zero-sized view.
Parameters
array_inThe 2D array that underlies this view.
row_rangeA begin, end pair that denote the extent of the rows in this view. The begin value corresponds to this view's zeroth element. The end value is one past the last element indexed by this view. To be valid, the end value must not be less than the begin value and must not be greater than the underlying vector's size.
col_inThe column of the array to be represented by this view. This must reference a valid column of the underlying array.
Exceptions
std::out_of_rangeif the range is invalid.
