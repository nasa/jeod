38 // PURPOSE:
39 //=============================================================================
40 // This simulation provides a reference implementation that can be used for
41 // comparison to other orbital dynamics implementations. It is intended to
42 // provide a reference set for dynamics comparison.
43 //
44 // This simulation models a single vehicle in orbit around the Earth. There
45 // are many adjustable configuration parameters that can be used to test out
46 // a suite of test cases with specific behavior. These test cases form the
47 // basis for simulation to simulation comparison.
48 //
49 // sys - Trick runtime executive and data recording routines
50 // time - Representations of different clocks used in the sim
51 // dynamics - Orbital dynamics
52 // env - Environment: ephemeris, gravity
53 // sun - Sun planetary model
54 // moon - Moon planetary model
55 // earth - Earth planetary model
56 // vehicle - Space vehicle dynamics model
57 //
58 //=============================================================================
