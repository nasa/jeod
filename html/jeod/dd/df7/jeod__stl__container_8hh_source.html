78 /**
79  * This is the base class for the JEOD replacements of the STL containers.
80  *
81  * A key goal of the JEOD STL container replacement effort is to
82  * provide checkpointable replacements that transparently provide the full
83  * functionality of the ISO/IEC 14882:2003 STL containers.
84  * This class begins that effort by defining types and member functions
85  * common to the STL deque, list, map, set, and vector class templates.
86  * Non-common methods are the responsibility of derived class templates
87  * specialized to a specific container types.
88  *
89  * @note Exceptions to full functionality goal:
90  * The above goal is not and never will be fully achieved. Exceptions are:
91  * - JEOD doesn't supply a replacement for std::deque or std::map.
92  * JEOD doesn't use deques at all and its maps are not checkpointable.
93  * - The full panoply of STL container constructors is not supplied.
94  * - The swap method is supplied but it is protected.
95  * The intent is that this class be further derived to create a
96  * checkpointable class. Swapping the checkpointable content is a dubious
97  * concept. The swap method is eventually exposed as the swap_stl_contents
98  * method to make it clear that that method is not a true swap.
99  */
