76 /**
77  * This is the base class for the JEOD replacements of the STL containers.
78  *
79  * A key goal of the JEOD STL container replacement effort is to
80  * provide checkpointable replacements that transparently provide the full
81  * functionality of the ISO/IEC 14882:2003 STL containers.
82  * This class begins that effort by defining types and member functions
83  * common to the STL deque, list, map, set, and vector class templates.
84  * Non-common methods are the responsibility of derived class templates
85  * specialized to a specific container types.
86  *
87  * @note Exceptions to full functionality goal:
88  * The above goal is not and never will be fully achieved. Exceptions are:
89  * - JEOD doesn't supply a replacement for std::deque or std::map.
90  * JEOD doesn't use deques at all and its maps are not checkpointable.
91  * - The full panoply of STL container constructors is not supplied.
92  * - The swap method is supplied but it is protected.
93  * The intent is that this class be further derived to create a
94  * checkpointable class. Swapping the checkpointable content is a dubious
95  * concept. The swap method is eventually exposed as the swap_stl_contents
96  * method to make it clear that that method is not a true swap.
97  */
