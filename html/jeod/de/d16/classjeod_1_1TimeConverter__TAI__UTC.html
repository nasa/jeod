◆ initialize_leap_second()

void jeod::TimeConverter_TAI_UTC::initialize_leap_second ( )
private

The conversion from Atomic Time (TAI) to Universal Time (UTC) involves the addition of leap seconds.

The number of leap seconds at any given (historical) time is provided in a table. This function initializes that table and sets the preliminary values.

Assumptions and Limitations
  • The table does not go into the future.
