void jeod::GravityManager::initialize_state ( const BaseDynManager &  manager)

Pass the initialize_state method to each GravitySource object registered with the gravity manager.

Assumptions and Limitations
  • Initialization phasing: The following must have been called prior to calling this method:
    • GravityManager::initialize_model to register the GravityManager object with the dynamics manager
    • GravityManager::add_grav_source to register all GravitySource objects in the simulation with the gravity manager.
    • Planet::register_model to associate the planet with a GravitySource.
    • DynamicsManager::activate_ephemerides to identify which reference frames can serve as integration frames.
