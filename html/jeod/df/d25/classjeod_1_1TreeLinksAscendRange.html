Non-default constructor.

Create a TreeLinksAscendRange given the start and end indices in the input Links object's path_to_node_ vector. Behavior is undefined if start_index > path_to_node_.size() or if end_index >= start_index.

Parameters
linksObject whose path_to_node_ vector is to be traversed, in reverse.
start_indexIndex of the element in the path_to_node_ vector that immediately follows the initial element to be visited in a range-based for loop. For example, using path_to_node_.size() starts at the final element of the vector.
end_indexIndex of the element in the path_to_node_ vector that is the last element to be visited in a range-based for loop. For example, using zero stops the iteration at the initial element in the vector.
